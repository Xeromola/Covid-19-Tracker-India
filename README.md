# Covid-19-Tracker-India

Created a Covid-19 Tracker using thw Govt of India API with The Django Framework and Bootstrap for frontend.

## How to run?
To install and run it on your own local machine you need to install Python Django and Requests module. You can do this by the following commands.

```
pip install django
pip install requests
```
After than you can open cmd/powershell/terminal in the same folder as manage.py and run the following command:-
```
python manage.py runserver
```
